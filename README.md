# Character Counter

A simple Character Counter web application built using HTML, Tailwind CSS, and JavaScript. This project allows users to type text into a textarea and view the character count in real time. The input field is automatically disabled once the maximum character limit is reached.

## 🚀 Live Demo

https://rajak1122.github.io/Character-Counter/

## ✨ Features

- Real-time character counting
- Displays the number of characters typed
- Maximum character limit validation
- Warning message when the limit is reached
- Automatically disables the textarea at the character limit
- Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript

## ⚙️ How It Works

1. The user enters text into the textarea.
2. JavaScript listens for input events.
3. The character count updates instantly.
4. When the maximum limit is reached:
   - A warning message is displayed.
   - The textarea is disabled to prevent additional input.
